# Calculadora de Matrices Interactiva 🧮

Una aplicación web puramente *frontend* diseñada para aprender, calcular y visualizar operaciones matemáticas con matrices en tiempo real.

## Descripción del Proyecto

Este proyecto nace para aprender 2 cosas muy importantes, el calculo de metrices y tambien aprender a usar *React* y *JavaScript* para crear aplicaciones web interactivas.

## Características Previstas

* **Operaciones Matemáticas:** Suma, resta, multiplicación y división (mediante matriz inversa).
* **Interfaz Dinámica:** Cuadrículas interactivas para introducir fácilmente los valores de las matrices de origen.
* **Cálculo Inmediato:** Visualización en tiempo real de la matriz resultado al interactuar con los números o cambiar el tipo de operación.
* **100% Frontend:** Toda la lógica y el procesamiento de datos se ejecutan de forma segura y rápida en el cliente. 

## Tecnologías

* **Core:** React + JavaScript
* **Build Tool:** Vite
* **Entorno de Desarrollo:** Visual Studio 2026
* **Hosting (Planificado):** Azure Static Web Apps
